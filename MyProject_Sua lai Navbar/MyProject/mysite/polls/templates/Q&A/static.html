Thiết lặp static file trong Django

  • Đầu tiên chúng ta tạo một thư mục có tên là static trong thư mục chính. Django sẽ tự động tìm các file tĩnh trong thư mục
  • Trong thư mục static này chúng ta lại tạo một file CSS có tên là style.css.Với nội dung tùy ý
  • Tiếp theo chúng ta tạo file .html minh họa. Trong đó khai báo
  • Sau đó chúng ta chỉ cần lấy biến static trong cặp thẻ ngoặc kép %% ra và gắn thêm đường dẫn đến các file css, js… của bạn
  • Dòng {% load staticfiles %} sẽ tự động gán đường dẫn đến thư mục static vào một biến có tên static do Django tự đặt, từ đó bạn chỉ cần lấy biến static là có thể lấy được đường dẫn tuyệt đối đến thư mục này.
